# restaurant-reservation
Restaurant Reservation with Vue.js and Node.js
## Requirements
- @vue/cli 5.0.8
- node 20.0.0
##
- Run backend by navigating to /backend
- Run frontend by navigating to /frontend

### Auth
- Login
- Register

### Customer
- Reserve Table
- Cancel Reservation

### Restaurant Owner
- Add new restaurant
- Update restaurant

### Admin
- Approve restaurant details
- Reject restaurant details
